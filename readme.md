# visia-filter (VISIA-style face analysis, approximation)

This repo generates a VISIA-like analysis panel from a standard photo:
- Spots
- Wrinkles
- Texture
- Pores
- Red Areas
- Porphyrins (approx)

> Note: VISIA UV/porphyrin imaging is hardware/calibration dependent. This is an approximation.
